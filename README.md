# AB__Programacion
## Clases Principales

### Archivos de Clase
- **Cita.h y Cita.cpp**: Clase para gestionar citas médicas
- **Medico.h y Medico.cpp**: Clase para gestionar información de médicos
- **Paciente.h y Paciente.cpp**: Clase para gestionar información de pacientes
- **SistemaHospital.h y SistemaHospital.cpp**: Clase principal que gestiona las operaciones del sistema
- **main.cpp**: Punto de entrada de la aplicación

## Características Principales del Sistema

### Funcionalidades
- **Gestión de Pacientes**
  - Alta, baja, modificación y búsqueda de pacientes
- **Gestión de Médicos**
  - Registro, especialidades y disponibilidad
- **Gestión de Citas**
  - Asignación, cancelación y modificación de citas
- **Generación de Reportes**
  - Informes sobre pacientes, citas y atenciones
- **Manejo de Archivos**
  - Guardado, carga y backup de datos

### Menú Principal
El sistema presenta un menú interactivo con las siguientes opciones:
1. Gestión de Pacientes
2. Gestión de Médicos
3. Gestión de Citas
4. Generación de Reportes
5. Manejo de Archivos

## Tecnologías
- **Lenguaje**: C++
- **Paradigma**: Programación Orientada a Objetos
- **Gestión de datos**: Vectores de la STL
- **Persistencia**: Archivos de texto

**Nota**: El código es un sistema hospitalario completo con múltiples funcionalidades de gestión.

## Requisitos
Para ejecutar este sistema, necesitas tener:

-C++11 o superior instalado en tu compilador
-Un sistema operativo compatible con C++
-Un editor de texto o IDE que soporte la sintaxis del lenguaje

## Uso
Para utilizar el sistema, sigue estos pasos:

1. Compile los archivos Cita.h, Medico.h, Paciente.h y SistemaHospital.cpp.
2. Ejecuta el archivo main.cpp para iniciar el menú principal.
3. Introduce una opción en la consola (Gestión de Pacientes, Gestión de Médicos, etc.) para acceder a las funcionalidades relevantes.
